# 指導

## 成為導師

導師計畫能讓雙方都有成長：導師能體驗對人負責的感覺，被指導者能得到一個專心致志的監督者。

導師的目標是建立良好有效的指導關係，並避免拖延到開發進度。指導實習生時，先規劃出要交付的專案（他們往往缺乏靈感），並切割成數個里程碑（解說需完成的事項並視對方反應調整）。這過程已開始培養成為主管的必要技能：

- 傾聽（擁有同理心的先決條件）：觀察對方回應，確認對方理解，並能問完所有問題
- 明確溝通（指出該發生的事）：提供指示和清晰思路，幫對方朝正確方向前進
- 調整回應：至少每周確認一次對方進度，即時地協助調整

指導新人的目標讓對方適應公司步調，並建立人脈網路。職涯成敗取決於拓展人脈的態度，這是項值得投入時間精力的投資。

對於成員間的技術或職涯指導，我們得明確了解自己的期望和目標。這段關係應讓彼此都有好處，導師應具體告知學員期望，並給予坦率建議；學員應為見面做準備，不浪費對方時間，並自問是否真需被指導，還只是需要一個可收費的專業人士的幫助。

## 主管與技術咖（Alpha Geek）

技術咖力求成為最優秀的工程師，但無能處理意見分歧，與接受他人出風頭。這樣的人不適宜被提拔到管理階層。

主管的任務是建立清晰、範圍明確、可衡量的目標：

- 釐清指導關係為何存在（幫對方跟上進度並提高生產力），確保計畫皆有指引和流程規劃
- 對指導關係會影響導師生產力有所準備
- 獎勵和培養團隊未來領導者。領導力存於彼此互動，培養耐心和同理心對任何人都很重要

## 給導師的關鍵要點

- 保持好奇和開放心態：過往經歷可能已限制固化我們的心態，指導關係提供我們培養好奇心和用新視角看待世界的絕佳機會，也幫助我們建立難得的人脈聯繫。
- 傾聽和使用對方語言：聽不進去發問，就不可能提供好答案。軟體開發是項團隊運動，有效溝通才能幫助團隊完成工作。
- 善待對方：職涯成敗取決於人脈強弱，指導關係是建立人脈的好方法

## 自我評估

- 公司是否有實習專案？若有，你願意指導實習生嗎？
- 公司對入職有哪些安排？你會指派導師給新人嗎？你會提議導師計畫，自願指導某人嗎？
- 你遇過優秀導師嗎？哪些做法讓你印象深刻？導師如何幫你學習－教會你什麼？
- 你有失敗的指導關係嗎？為何不盡人意？你汲取了哪些教訓，避免再度發生？
