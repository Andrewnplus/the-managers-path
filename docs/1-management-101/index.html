                      管理入門課 #

                      對主管的期待 #

                      個人的管理哲學皆從「被管理」所累積，良好工作關係始於與主管的一對一會議（1-on-1），這有兩個好處：

                      與主管建立人際關係：優秀管理者會注意到你的工作精力有變化，並主動關心 能定期私下談需討論的事：我們得以提前規劃，主動提出需求 　　此外要接收回饋：越早知道自身壞習慣，就越易即時修正。主管應能提供資源，助你改善，可用尋求建議來表明這份尊重。重點要勇於開口：無論是升職加薪或對某成員有不滿，都得主動告知才會得到對應舉動。

                      主管應清楚你是否有資格晉升：他會知道公司需要什麼樣的人才，並助你取得必要技能和成就。以目標成為 CTO 為例，除了得具有良好技術能力，還得會強大的溝通技巧、專案管理技能和對產品的深刻認識。所以我們可以觀察優秀的經理和導師如何工作，學習他們的「工作方法」。

                      如何被管理 #

                      要擁有幸福職涯，得為培養工作經歷具備責任感：主管無法主動知道什麼會讓你開心，而每個五至十年，我們總會經歷對於職涯的茫然與不確定性。我們得了解自己：確定下一步在哪、視主管為資源，助你發掘自身可能性。

                      接著我們得追求內心渴望：主動爭取機會、尋求其他資源、主動尋求回饋。過程可能不有趣或舒適，但會是最快的前進策略。維護與主管的關係時得注意：能改變的只有自己、聚焦於提供解案，用尋求建議表達尊重和展現信任。

                      主管的好壞深遠影響你的職涯：強大主管知道如何在公司打交道－讓你順利升職，助你得到重要人物的關注和回饋；他們的強大人脈，能讓你離職後也會遇到新機運。許多優秀工程師最終淪為平庸經理，是因他們不知道或根本不想處理管理階層的政治角力。

                      自我評估 #

                      • 你遇過好主管嗎？他做過哪些難能可貴的事？
                      • 你多久與主管 1on1？你會主動提及話題嗎？
                      • 生活重大變化時，你敢告訴主管嗎？他了解你多深？
                      • 主管是否會給回饋？好的、壞的、或從來沒有？
                      • 主管有幫你訂定今年的工作目標嗎？
